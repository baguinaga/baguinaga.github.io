Ninja Bomber

Ninja Bomber is an HTML5/JS game built using the Phaser framework. The objective is to collect as many stars without touching randomly moving bombs. Ninja Bomber is a full-stack app with user authentication and player leaderboards.

Star Wars RPG

Star Wars RPG is a Star Wars themed game built using jQuery. Perhaps better described as an RPG-lite. The objective of the game is to defeat all the characters with the selected player character in the correct order.
